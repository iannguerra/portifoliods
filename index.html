Projetos

Análise de dados com Python

Neste projeto, conceitos de programação em Python, manipulação de dados, raciocínio estratégico e lógica de negócios, juntamente com ferramentas de desenvolvimento web como Streamlit e Github, foram empregados para criar um painel interativo que forneça métricas e visualizações estratégicas para entender melhor a distribuição geográfica dos restaurantes, características dos países, informações sobre cidades, estatísticas de restaurantes e insights sobre tipos de culinária. O resultado final do projeto foi um painel hospedado em um ambiente de nuvem e acessível por meio de um link da web.

Ferramentas utilizadas: - Python, Jupyter Lab, Streamlit e GitHub
